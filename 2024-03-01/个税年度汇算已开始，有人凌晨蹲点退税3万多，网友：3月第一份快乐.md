# 个税年度汇算已开始，有人凌晨蹲点退税3万多，网友：3月第一份快乐

三月第一天，大家被同一个消息刷屏，那就是退税！

****3月1日开始，2023年度个税汇算可以办理了，汇算的时间为3月1日至6月30日，依然是“多退少补”。

在社交平台上，不少网友称“终于等到这一天”“凌晨就开始蹲点了，奔跑在退税前线”。

据网友们晒出的退税结果，几百、几千、几万的都有，有网友直呼“喜提一笔巨款”“三月的第一份快乐”。

![](https://inews.gtimg.com/om_bt/OLsrfJmQPOGD8XDDVPL-
jzSIqmIuZivekeb6frukBdTwoAA/1000)

不过，也有不少不退不补和需要补税的网友，“激动地打开个税APP，退税金额：0”“退税失败，甚至还得补”“本想着能到手一笔巨款，结果却要付一笔巨款”。

截至发稿，个税APP显示，3月1日、2日、3日预约已满。后续日期仍然可以预约。

![](https://inews.gtimg.com/om_bt/Oinh0kiXI-
mQkzXOJcAfb1ell6vxfnxQvAoI5TAMrTpDEAA/1000)

需要注意的是，**个税汇算办理时间为2024年3月1日至6月30日，纳税人如需在3月1日至20日之间办理汇算，可在2月21至3月20日（每日6:00-22:00）通过个税APP预约。3月21日至6月30日，纳税人无需预约，可以随时办理。**

此外，税务总局近日发布公告明确，一些项目可在个税汇算期间填报或补充扣除。
