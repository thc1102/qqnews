# 恒指跌0.36%恒科指涨0.99%，理想汽车涨21%

金融界2月27日消息
今日上午，港股主要指数走势分化，截至午间，恒生指数跌0.36%，报16574.21点，恒生科技指数涨0.99%，报3426.96点，国企指数涨0.07%，报5727.29点，红筹指数跌1.07%，报3460.36点。

大型科技股中，阿里巴巴-SW涨0.2%，腾讯控股跌2.3%，京东集团-SW跌1.88%，小米集团-W涨0.92%，网易-S跌0.78%，美团-
W跌1.79%，快手-W跌2.5%，哔哩哔哩-W跌0.95%。

业绩大超预期，理想汽车大幅飙涨21%，亦带领汽车股集体走强，比亚迪股份、小鹏汽车、蔚来汽车皆上涨，锂电池股、特斯拉概念股等齐涨。

数字货币概念股齐涨，OSL集团涨超19%。消息方面，比特币涨破54000美元，创下逾两年新高，现报54674.5美元/枚。此外Fundstrat
Global Advisors管理合伙人Tom Lee对比特币走势做出了激进预测，称到今年年底，这一加密货币可能会攀升至15万美元。

机构指全球半导体制造业将于2024年复苏，半导体股表现活跃，龙头中芯国际涨近5%。

家电股、影视娱乐股、煤炭股多数上涨。

春节假期过后日均入境旅客回落至约8万人次，濠赌股跌幅靠前。

昨日领衔上涨的物管股集体回调，内房股继续走低。

手游股、光伏股、重型机械股、互联网医疗股普遍下跌。

新秀丽一度涨超18%。报道称新秀丽在收到潜在买家的收购意向后，正在考虑各种选择，包括私有化在内的可能性。

东方甄选跌4.91%，消息面上，董宇辉微博账号内容已全部清空。

鼎益丰一度暴跌超40%，截至午间收盘股价跌20.83%。消息面上，2月26日，香港证券及期货事务监察委员会在官网发布新闻稿称，香港证监会已在原讼法庭对鼎益丰控股集团国际有限公司前主席兼非执行董事隋广义及另外20名人士展开法律程序，指他们涉嫌在2018年3月1日至2018年9月14日期间内操纵鼎益丰的股份。
